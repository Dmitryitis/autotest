# Autotest

## Описание

Проект по автоматизации тестов

## Стек технологий

Python, pytest, playwright, test-it

## Install dependencies
```
python -m venv env
pip install -r requirements.txt
```

### Activate environment
```
./env/Scripts/activate
```

### Run tests
```
pytest
```

### Create report
```
./generate_report.sh
```

### Upload result in test-it
```
./testit.sh
```
